The `MonetizationCurrencyAmount` interface is used to supply information about monetary amounts. It maps directly to the <LinkOut href="https://www.w3.org/TR/payment-request/#dom-paymentcurrencyamount">Payment Request API</LinkOut> `PaymentCurrencyAmount` dictionary.

```js
WebIDL[SecureContext, Exposed=Window]
interface MonetizationCurrencyAmount {
  readonly attribute DOMString currency;
  readonly attribute DOMString value;
};
```
