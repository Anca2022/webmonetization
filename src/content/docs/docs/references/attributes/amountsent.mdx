The `MonetizationEvent` `amountSent` attribute returns the amount and currency code of the money that was successfully sent in the last payment.

## Value

A dictionary with the same data structure as the <LinkOut href="https://www.w3.org/TR/payment-request/#dom-paymentcurrencyamount">`PaymentCurrencyAmount`</LinkOut> dictionary in the <LinkOut href="https://www.w3.org/TR/payment-request/">Payment Request API</LinkOut>. The returned value is the value it was initialized with.

## Example

```html
<link rel="monetization" href="https://wallet.example/alice" />
<script>
  const link = document.querySelector('link[rel="monetization"]')
  link.addEventListener('monetization', (event) => {
    // See how much was sent and in what currency
    const {
      amountSent: { value, currency },
    } = event
    console.log(`Browser sent ${currency} ${value}.`)
  })
</script>
```
