The `MonetizationEvent` `paymentPointer` attribute returns a URL representing the payment endpoint.

## Value

A URL representing the payment pointer included within the monetization `<link>` element. The returned value is the value it was initialized with.

## Example

```html
<link rel="monetization" href="https://wallet.example/alice" />
<script>
  const link = document.querySelector(“link[rel=‘monetization’]“);
  link.addEventListener(“monetization”, (event) => {
  console.log(event.paymentPointer)})
</script>
```
