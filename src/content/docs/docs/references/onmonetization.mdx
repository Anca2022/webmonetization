The `onmonetization` attribute of the <Tooltip content='Describes event handlers common to several interfaces like HTMLElement, Document, or Window' client:load><span>`GlobalEventHandlers`</span></Tooltip> mixin is used to process `monetization` events sent to the `Window`.

`onmonetization` can be applied to any element and is used by the user agent (such as a web browser or WM agent) to notify that some `<link>` element is monetized.

## Syntax

```js
window.onmonetization = functionRef
```

## Example

Show a visitor an alert box and message when a Web Monetization event is triggered.

```js
window.onmonetization = function () {
  alert('Web Monetization event triggered.')
}
```
